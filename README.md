# Periodic Table Hackathon Project

## Description
An interactive **Periodic Table** web application built for a hackathon.  
Explore chemical elements with dynamic details like atomic number, symbol, properties, and classifications. Perfect for learning and quick reference.

## Features
- Interactive element cards with details  
- Search and filter elements by name, symbol, or group  
- Color-coded groups for easy visualization  
- Responsive and user-friendly interface  

## Tech Stack
- Frontend: HTML, CSS, JavaScript.
